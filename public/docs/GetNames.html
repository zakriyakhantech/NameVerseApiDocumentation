Get Names

GET /api/names

Retrieve a filtered list of names with pagination, cultural filters, and lucky attributes.

Description

This endpoint returns a paginated list of names based on comprehensive filtering criteria. It supports filtering by starting alphabet (A-Z), gender (boy/girl), religious/cultural origin (Christian, Islamic, Hindu), language, category, theme, and lucky attributes including lucky day, color, and stone.

Query Parameters

Pagination

page integer optional

Page number for pagination.

Default: 1
limit integer optional

Maximum number of names to return per page.

Default
20
Minimum
1
Maximum
500

Basic Filters

gender enum optional

Filter names by gender.

Allowed Values
boy girl
alphabet string optional

Filter names by their starting letter. Case-insensitive, accepts any letter A-Z.

Examples
A M Z

Cultural Filters

religion enum optional

Filter names by religious or cultural origin.

Allowed Values
christian islamic hindu
origin string optional

Filter names by cultural origin.

Examples
Arabic Greek Latin
language string optional

Filter names by language origin.

Examples
Hebrew Sanskrit

Thematic Filters

category string optional

Filter names by category.

Examples
Modern Traditional Classic
theme string optional

Filter names by thematic meaning.

Examples
Nature Strength Wisdom

Lucky Attributes

luckyDay string optional

Filter names by lucky day of the week.

Examples
Monday Friday
luckyColor string optional

Filter names by lucky color.

Examples
Blue Red Green
luckyStone string optional

Filter names by lucky gemstone.

Examples
Ruby Sapphire Emerald

Request Example

GET /api/names?page=1&limit=20&gender=boy&religion=islamic&alphabet=M
