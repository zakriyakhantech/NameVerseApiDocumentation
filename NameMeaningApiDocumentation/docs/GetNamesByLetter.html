Get Names by Letter

Live Testing
GET /api/name/letter/{letter}

Retrieve names by starting letter with optional filters: religion, gender, theme, sorting, pagination, and items per page.

Description

This endpoint returns a paginated list of names starting with a specific letter. You can filter results by religion, gender, and theme, sort by popularity, and control pagination using page and perPage parameters.

Query Parameters

letter string required

The starting letter to filter names by (A-Z, case-insensitive).

religion enum optional

Filter names by religion.

christian islamic hindu
gender enum optional

Filter names by gender.

boy girl
theme string optional

Filter names by theme, e.g., Beauty, Strength, Nature.

sort enum optional

Sort names by criteria like popularity.

popularity alphabetical
page integer optional

Page number for pagination.

perPage integer optional

Number of results per page (max 50).

Request Example

GET https://namverse-api.vercel.app/api/name/letter/r?religion=christian&gender=girl&theme=Beauty&sort=popularity&page=1&perPage=50

Response

200 Success 
{
  "success": true,
  "page": 1,
  "perPage": 50,
  "total": 120,
  "data": [
    "Rachel",
    "Rania",
    "Rebecca",
    "Renee",
    "Rita"
  ]
}
400 Bad Request 
{
  "error": "Invalid letter or query parameters"
}
