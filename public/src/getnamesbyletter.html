NameVerse API

Get names by starting letter with filters

Health Check

Endpoint

GET /api/name/letter/{letter}

Get names starting with a specific letter, with optional filters for religion, gender, theme, sorting, and pagination

Path Parameters

letter required

Starting letter (A-Z)

a, b, c, ..., z

Query Parameters

religion optional

Filter by religion

christian islamic hindu
gender optional

Filter by gender

boy girl
theme optional

Filter by theme (e.g., Beauty, Strength, Wisdom)

sort optional

Sort results

popularity alphabetical
page optional

Page number (default: 1)

perPage optional

Results per page (default: 50)

Examples

All names starting with 'R'

/api/name/letter/r

Christian girl names with 'R'

/api/name/letter/r?religion=christian&gender=girl

Beauty themed, sorted by popularity

/api/name/letter/r?theme=Beauty&sort=popularity

Test API

/api/name/letter/r?religion=christian&gender=girl&theme=Beauty&sort=popularity&page=1&perPage=50
