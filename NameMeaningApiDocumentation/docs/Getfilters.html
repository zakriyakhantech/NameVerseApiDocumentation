Get Filter Options

Live Testing
GET /api/filters

Retrieve all available filter options including religions, genders, themes, origins, and alphabets for filtering names.

Description

This endpoint returns all available filter options that can be used across the API. It provides the complete list of religions, genders, themes, origins, and alphabet letters that you can use to filter name queries. This is particularly useful for building dynamic filter UIs and ensuring your filters match the available options.

Query Parameters

No parameters required

This endpoint does not accept any query parameters. Simply call the endpoint to get all filter options.

Request Example

GET https://namverse-api.vercel.app/api/filters

Response Fields

religions

Array of available religion filters with counts

genders

Array of available gender filters with counts

themes

Array of available theme categories with counts

origins

Array of available name origins with counts

alphabets

Array of all available starting letters (A-Z)
