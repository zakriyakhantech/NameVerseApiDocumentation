Live Testing
GET /api/names/search

Search names by keyword, substring, or partial text matching.

Description

This endpoint allows searching for names using a text query. It supports partial matches, case-insensitive search, and returns a list of all names matching the text pattern. Works for all religions and alphabets.

Query Parameters

query string required

Text keyword to search names by. Supports partial search.

Examples
aa rah mu
limit integer optional

Maximum number of search results to return.

Default
50
Minimum
1
Maximum
500

Request Example

GET /api/names/search?query=aa&limit=10

Response

200 Success 
{
  "success": true,
  "count": 3,
  "data": [
    "Aarav",
    "Aayan",
    "Aamina"
  ]
}
400 Bad Request 
{
  "error": "Missing required parameter: query"
}
