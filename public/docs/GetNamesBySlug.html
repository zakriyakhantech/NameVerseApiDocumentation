Description

This endpoint retrieves detailed information for a specific name using its slug. The slug is a unique, SEO-friendly string such as abdul-rauf.

Path Parameter

slug string required

The unique slug of the name. Must be lowercase and hyphen-separated.

Examples
abdul-rauf muhammad-ali sara-johnson

Request Example

GET /api/names/abdul-rauf

Response

200 Success

    "gender": "Male",

  }
}
