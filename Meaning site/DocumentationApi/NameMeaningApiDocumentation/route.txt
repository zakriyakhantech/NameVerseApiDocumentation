===========================================
TESTING TIPS FOR POSTMAN
===========================================

1. Create a Collection called "NameVerse API"

2. Create Environment Variables:
   - baseUrl: http://namverse-api

3. Use {{baseUrl}} in your requests:
   Example: {{baseUrl}}/api/names

4. Test Response Codes:
   - 200: Success
   - 400: Bad Request
   - 404: Not Found
   - 500: Server Error

5. Save common requests as examples

6. Use Postman Tests to validate responses:
   pm.test("Status code is 200", function () {
       pm.response.to.have.status(200);
   });

7. Check response structure and data types
